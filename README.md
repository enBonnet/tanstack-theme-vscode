# Description

This is the non-official TanStack Theme for Visual Studio Code.

## Inspiration

This theme is inspired by the [TanStack](https://tanstack.com/) brand colors, and based on the [Dracula Theme](https://draculatheme.com/) schema of colors.

# Development

To develop this theme, you will need to have [Node.js](https://nodejs.org/) installed.
